# 기술문서 목차

Arduino Nano + nRF24L01+ 기반 DIY RC 항공기 송수신기의 구현 원리를 주제별로 설명하는 문서 모음입니다.

조이스틱 입력 처리부터 RF 전송, ESC·서보 구동, 신호 안정화까지 각 단계의 설계 결정 이유와 핵심 수식·파라미터 값을 구체적으로 기술합니다.

---

<br>

## 읽는 순서

처음 접하는 경우 아래 순서를 권장합니다.

```mermaid
flowchart LR
    D01["01<br>하드웨어"] --> D02["02<br>RF 통신"]
    D02 --> D03["03<br>조이스틱 매핑"]
    D03 --> D04["04<br>송신기 루프"]
    D04 --> D05["05<br>ESC·서보"]
    D05 --> D06["06<br>신호 안정화"]
```

하드웨어 연결(01)과 RF 통신(02)으로 전체 구조를 파악한 뒤, 송신기(03·04)와 수신기(05·06)를 순서대로 읽으면 데이터 흐름을 자연스럽게 따라갈 수 있습니다.

## 문서별 한줄 요약

1. **[하드웨어 구성 및 핀 매핑](./01-hardware.md)**  
   Arduino Nano 두 대에 nRF24L01+, 조이스틱 2개, ESC, 서보를 연결하는 방법과 전원 주의사항을 설명합니다.

2. **[nRF24L01+ 무선 통신](./02-rf-communication.md)**  
   채널 108·250kbps·Auto-ACK 설정과 4바이트 Signal 구조체를 50Hz로 주고받는 방법을 설명합니다.

3. **[조이스틱 캘리브레이션과 채널 매핑](./03-joystick-mapping.md)**  
   ADC 값(0~1023)을 채널별 특성(반전·데드존·좌우 대칭)에 맞게 0~255 byte로 변환하는 방법을 설명합니다.

4. **[송신기 전송 루프](./04-transmitter.md)**  
   `millis()` 비교로 정확한 50Hz 주기를 유지하고, 연속 실패 시 라디오를 자동 재초기화하는 방법을 설명합니다.

5. **[ESC와 서보 제어](./05-esc-servo.md)**  
   수신된 byte 값을 ESC PWM 펄스(1510~2000µs)와 서보 각도(30~150°)로 변환하여 모터와 조종면을 구동하는 방법을 설명합니다.

6. **[신호 안정화](./06-signal-stability.md)**  
   스무딩 필터·데드밴드·페일세이프·자동 절전·천천히 움직이기를 조합하여 서보 떨림을 제거하고 신호 끊김을 안전하게 처리하는 방법을 설명합니다.
